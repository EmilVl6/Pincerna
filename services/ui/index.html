About

Pincerna Cloud — personal device management and file access. This interface is a lightweight control panel for your Raspberry Pi. Use Files to browse storage, and VPN to connect your device to your home network.

Security: authentication is required for protected actions. Do not expose direct file shares to the public internet — use the VPN for full network access.

Current features: demo sign-in, file browser UI (requires backend API), VPN UI toggle (placeholder), and device controls (restart, metrics).
